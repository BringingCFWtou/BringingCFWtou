Panasonic DMP-BDT310EG // Region-Free CFW Guide

Komplette Dokumentation für Custom Firmware und Region-Hacks auf dem Panasonic DMP-BDT310EG (Blu-ray Player, Europa-Modell).

[ WARNUNG ] — Das Modifizieren deines Gerätes kann die Garantie erlöschen lassen. Notiere vorher Firmware-Version und MAC-Adresse. Du machst alles auf eigene Verantwortung.
Gerät & Firmware-Info
EigenschaftWert
ModellPanasonic DMP-BDT310EG
Region ab WerkBlu-ray Region B  /  DVD Region 2
Firmware anzeigenSetup → System → Firmware-Version
LAN-MAC findenSetup → Netzwerk → Netzwerkkonfiguration
(ggf. von WLAN auf LAN umschalten)
Modell-VariantenDMP-BDT310-EG-K  /  EG-S  /  EG-W
[ INFO ] — "EG" steht für Europa/UK/CIS. Dieser Guide gilt ausschließlich für EG-Modelle.
Methode 1: Region-Free CFW via CD Empfohlen

Modifizierte Firmware wird per CD auf den Player geflasht. Schaltet dauerhaft alle DVD- und Blu-ray-Regionen frei. Benötigt eine personalisierte PANA_DVD.FRM Datei die für dein spezifisches Gerät erstellt wird.

[ BENÖTIGT ]
• Aktuelle Firmware-Version
• LAN-MAC-Adresse (nicht WLAN-MAC!)
• Genaue Modellbezeichnung (z.B. DMP-BDT310-EG-K)
• Leerer CD-R oder CD-RW + CD-Brenner

Schritt-für-Schritt

  1. Infos sammeln:
    Setup → Netzwerk → Netzwerkkonfiguration: LAN-MAC notieren.
    Setup → System: Firmware-Version notieren.
    Modell-Buchstaben auf Geräterückseite nachschauen.
  2. CFW anfordern:
    Kontaktiere den CFW-Anbieter (z.B. highdefinition.ch) mit Modell, MAC und Firmware-Version. Du bekommst eine personalisierte .zip-Datei.
  3. PANA_DVD.FRM auf CD brennen:
    ZIP entpacken. PANA_DVD.FRM direkt ins Root der CD brennen — nicht in einen Unterordner.
    Windows: Rechtsklick auf die .FRM-Datei → "Senden an" → CD/DVD-Laufwerk.
  4. Firmware flashen:
    CD in den Player einlegen. Player erkennt die Firmware automatisch. WÄHREND des Flashens niemals Strom trennen! Player startet danach neu.
  5. Online-Updates deaktivieren:
    Setup → Netzwerk → Firmware-Update: AUS. Sonst überschreibt ein offizielles Update den Mod.
[ ERGEBNIS ] — Player spielt jetzt DVD Region 1–6 und Blu-ray A/B/C ab. Mod ist dauerhaft.
Setup → Netzwerk → Firmware-Update → AUS → Mit OK bestätigen
Methode 2: IR-Hack via Universal Remote Ohne CFW

Per programmiertem Universal-Remote (Logitech Harmony oder One for All) wird eine IR-Sequenz gesendet, die den Service-Modus aktiviert. Kein Flashen notwendig.

[ BESTÄTIGT ] — Funktioniert auf DMP-BDT310EG mit Harmony 880 und Harmony One. Schaltet DVD 1–6 und Blu-ray A+B+C frei.

IR-Sequenz (10 Codes)

Je 500ms Pause zwischen den Codes. Reihenfolge genau einhalten!

1ENTER00191

4<< (Rücklauf)00180

Harmony-Setup

  1. Gerät "Panasonic DMP-BDT310" in Harmony Software hinzufügen.
  2. Jeden der 10 Hex-Codes als eigenen Befehl anlegen.
  3. Neue Activity erstellen, alle 10 Befehle in Reihenfolge mit je 500ms Pause.
  4. Player einschalten, Activity starten. Automatischer Neustart = Erfolg.
[ TIPP ] — Falls Activity nicht klappt: Buttons am Touchscreen manuell nacheinander drücken. Hat bei mehreren Nutzern besser funktioniert.

Blu-ray Bypass-Trick (ohne Remote)

1. Disc einlegen → "Falsche Region" abwarten 2. STOP drücken 3. 1 → ENTER (oder OK) 4. Bei Fehler: Schritt 2–3 wiederholen
FAQ

Q: Welche Methode soll ich nehmen?

Dauerhaft alle Regionen → Methode 1 (CFW). Kein CD-Brenner oder nur DVDs nötig → Methode 2 (IR-Hack).

Q: Brick-Risiko?

IR-Hack: praktisch kein Risiko. CFW-Flash: gering, solange CD korrekt gebrannt und kein Stromabbruch. Trotzdem auf eigene Verantwortung.

Q: Sub-Modell unbekannt (EG-K, EG-S...)?

Typenschild auf der Geräterückseite prüfen. Alternativ den CFW-Anbieter mit der Seriennummer kontaktieren.

Q: Normale Discs funktionieren noch?

Ja. CFW ändert nur die Region-Beschränkung, alles andere bleibt unverändert.

Q: Harmony Activity klappt nicht.

Buttons am Touchscreen manuell nacheinander drücken, 500ms Pause einhalten.
